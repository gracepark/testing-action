# Viewing your GitHub Copilot usage

You can view how many users have access to GitHub Copilot across your organization.

**Who can use this feature**: Organization admins can view usage for GitHub Copilot in their organization.
GitHub Copilot can be managed through personal accounts with GitHub Copilot for Individuals or through organization accounts with GitHub Copilot for Business.

## About your GitHub Copilot usage

You can view usage information for GitHub Copilot for Business in your  organization, broken down by seat assignment status.  At the organization level, this information includes the total number of seats, seats carried over from the previous billing cycle, new seats added during the current cycle, and seats to be removed at the end of the current cycle.
